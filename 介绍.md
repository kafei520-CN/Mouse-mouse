# Mouse&mouse / 鼠标分流

## 中文

Mouse&mouse 是一个面向 Minecraft 1.21.1 NeoForge 的客户端输入分流模组，适合多窗口、多套键鼠、本地协作游玩或输入测试场景。

它可以通过 Windows Raw Input 识别不同的键盘和鼠标设备，并允许你为不同的 Minecraft 实例选择独立的输入设备。这样，多开窗口时，每个窗口都可以接收指定设备的输入，减少多个实例抢占同一套键鼠的问题。

### 主要特性

- 支持为不同 Minecraft 窗口选择不同的键盘 / 鼠标设备
- 使用虚拟光标处理菜单界面交互
- 支持世界内视角、点击、滚轮和键盘输入分流
- 通过 `Alt + F8` 打开设备选择界面
- 设备选择只在当前运行实例中生效，重启后需要重新选择，避免设备 ID 变化导致误选

### 关于内置的 splitter.exe

本模组内置了一个名为 `splitter.exe` 的 Windows 辅助程序。它是模组正常工作所必需的 Raw Input 分流器，用于读取本机键鼠设备输入，并通过本机回环地址 `127.0.0.1` 与 Minecraft 客户端通信。

它不是病毒、广告程序、挖矿程序或远程控制程序；不会上传文件、不会收集账号信息，也不会连接外部服务器。它的作用仅限于在本机识别键鼠设备并把选中的输入发送给当前 Minecraft 实例。

由于该功能依赖 Windows Raw Input，目前仅支持 Windows 客户端。

## English

Mouse-mouse is a client-side input-splitting mod for Minecraft 1.21.1 NeoForge. It is designed for local multi-instance play, multiple keyboard/mouse setups, local co-op experiments, and input testing.

The mod uses Windows Raw Input to detect individual keyboard and mouse devices, then lets each Minecraft instance claim its own selected devices. This helps multiple game windows receive separate input instead of fighting over the same keyboard and mouse.

### Features

- Select different keyboard / mouse devices for different Minecraft windows
- Virtual cursor support for menu screens
- Routed mouse movement, clicks, scrolling, and keyboard input in-game
- Open the device selection screen with `Alt + F8`
- Device selection is temporary per running instance, reducing wrong-device issues after hardware or driver changes

### About the bundled splitter.exe

This mod includes a bundled Windows executable named `splitter.exe`. It is required for the mod’s Raw Input functionality. The program reads local keyboard and mouse input through Windows Raw Input and communicates with Minecraft only through the local loopback address `127.0.0.1`.

It is not a virus, adware, miner, or remote-control tool. It does not upload files, collect account information, or connect to external servers. Its only purpose is to identify local input devices and forward the selected device input to the current Minecraft instance.

Because this feature depends on Windows Raw Input, the mod currently supports Windows clients only.
